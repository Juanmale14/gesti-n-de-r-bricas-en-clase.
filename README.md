❄️ Gestión de Rúbricas de Evaluación

Una herramienta web moderna, intuitiva y visualmente ligera diseñada para docentes que necesitan agilizar el proceso de evaluación por rúbricas. Esta aplicación permite cargar criterios personalizados, evaluar alumnos en tiempo real y obtener análisis estadísticos inmediatos con un diseño pastel azul profesional.

🚀 Características Principales

Interfaz Azul Pastel: Diseño limpio y relajante centrado en la usabilidad y la reducción de la fatiga visual.

Carga de Rúbricas Inteligente: Soporta archivos Excel (.xlsx) y Markdown (.md). Incluye un sistema de limpieza automática para ignorar filas vacías accidentales al final de los archivos.

Drag & Drop: Área de carga de archivos mejorada con soporte para arrastrar y soltar de forma fluida.

Evaluación Interactiva: Tabla dinámica donde se selecciona el nivel de logro con un clic, calculando la nota final instantáneamente.

Análisis de Datos: - Gráficos de barras con la media grupal por indicador.

Alertas visuales: Las notas inferiores a 5.0 se resaltan automáticamente en rojo intenso para identificar alumnos con dificultades.

Exportación Versátil: Genera informes detallados en PDF (manteniendo el resaltado de suspensos) o datos puros en CSV para Excel.

Privacidad Local: Los datos se guardan en el localStorage del navegador, garantizando que la información no salga del dispositivo del docente.

🛠️ Tecnologías Utilizadas

HTML5 & CSS3: Maquetación moderna con Flexbox, Grid y animaciones suaves.

JavaScript (Vanilla): Lógica de la aplicación y gestión de estados.

XLSX (SheetJS): Para el procesamiento de hojas de cálculo.

jsPDF & AutoTable: Para la generación de informes profesionales en PDF.

Google Fonts: Uso de la tipografía 'Quicksand' para una mejor legibilidad.

📖 Instrucciones de Uso

Configuración: Sube tu propia rúbrica arrastrando un archivo Excel o Markdown. La primera columna debe contener los indicadores y las siguientes los niveles de logro (10, 8, 6, 5, 1).

Evaluación: Introduce el nombre del estudiante y marca los casilleros correspondientes. Haz clic en "Registrar Evaluación".

Análisis: Consulta la pestaña de análisis para ver cómo va el grupo en general y quién necesita refuerzo.

Exportar: Al finalizar el periodo, descarga el informe en PDF para tu archivo personal o un CSV para integrarlo con otras plataformas.
